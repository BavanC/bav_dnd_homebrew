D&D Session 0 Questionnaire - Public
This is a pre-session 0 questionnaire to gather information about player expectations. The DM will see the answers to the questions, but no one else will.

The purpose of this questionnaire is to enrich the gaming experience for everyone involved, and will provide the DM with the tools to do so.

Once all the responses are in, we can discuss as a group. And then the fun can begin!

(this is the public version, just to gather answers for curiosity's sake. For a real game, I would make a copy and send this out to only the players.)
